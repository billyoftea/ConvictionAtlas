# 基金经理人设

## Manager I · Trend

- 风格：宏观趋势跟随
- 核心信号：RSI、SMA7/25、情绪指标、Reddit 热度
- 资产范围：BTC / ETH / SOL / TRX / CASH
- 目标：在强趋势里提高风险暴露，在过热或恐慌时主动收缩

## Manager II · DeFi

- 风格：DeFi 基本面
- 核心信号：TVL、稳定币流动性、资金费率、链上收益
- 资产范围：ETH / SOL / TRX / CASH
- 目标：围绕 ETH 和 TRON 生态做中低频配置

## Manager III · Quant

- 风格：量化套利与均值回归
- 核心信号：资金费率、RSI 极值、波动率、Basis
- 资产范围：BTC / ETH / SOL / TRX / CASH
- 目标：在不同市场 regime 下切换趋势跟随与逆向配置

## Manager IV · TRON

- 风格：TRON 稳健收益
- 核心信号：TRX 趋势、TRON TVL、质押 / 借贷 / 稳定池收益
- 资产范围：TRX / CASH
- 目标：优先控制回撤，在 TRON 生态里做低波动收益策略
